## –û—Ç—Å—Ç—É–ø—ã (indentation)

–û—Ç—Å—Ç—É–ø—ã —ç—Ç–æ –≤–∞–∂–Ω–∞—è –∫–æ–Ω—Ü–µ–ø—Ü–∏—è —è–∑—ã–∫–∞ Python.

- –û—Ç—Å—Ç—É–ø—ã —ç—Ç–æ —Å–ø–æ—Å–æ–± —Å–∫–∞–∑–∞—Ç—å –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä—É, —á—Ç–æ –æ–ø–µ—Ä–∞—Ü–∏—è (–∏–ª–∏ –≥—Ä—É–ø–ø–∞) —ç—Ç–æ –æ–¥–∏–Ω –±–ª–æ–∫ –∫–æ–¥–∞
- –ü—Ä–æ–≥—Ä–∞–º–º–Ω—ã–µ –±–ª–æ–∫–∏ –≤ Python –æ—Å–Ω–æ–≤–∞–Ω—ã —Ç–æ–ª—å–∫–æ –Ω–∞ –æ—Ç—Å—Ç—É–ø–∞—Ö.
- –†–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å 4 –ø—Ä–æ–±–µ–ª–∞ –¥–ª—è –∫–∞–∂–¥–æ–≥–æ —É—Ä–æ–≤–Ω—è –≤–ª–æ–∂–µ–Ω–Ω–æ—Å—Ç–∏ (—Ç–∞–±—É–ª—è—Ü–∏—è —Ä–∞–∑—Ä–µ—à–µ–Ω–∞, –Ω–æ –Ω–µ —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è)
- –ù–µ–ª—å–∑—è —Å–º–µ—à–∏–≤–∞—Ç—å –≤ –æ–¥–Ω–æ–º —Ñ–∞–π–ª–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –ø—Ä–æ–±–µ–ª–æ–≤ –∏ —Ç–∞–±—É–ª—è—Ü–∏–∏

```python
# [0 –ø—Ä–æ–±–µ–ª–æ–≤] - –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏, "–∫–æ—Ä–Ω–µ–≤–æ–π" —É—Ä–æ–≤–µ–Ω—å
def get_abs(a):
    # [4 –ø—Ä–æ–±–µ–ª–∞] - –≤—Å–µ, —á—Ç–æ –æ—Ç–Ω–æ—Å–∏—Ç—Å—è –∫ —Ñ—É–Ω–∫—Ü–∏–∏ –±—É–¥–µ—Ç –Ω–∞—á–∏–Ω–∞—Ç—å—Å—è, –∫–∞–∫ –º–∏–Ω–∏–º—É–º –≤ 4 –ø—Ä–æ–±–µ–ª–æ–≤
    # –µ—Å–ª–∏ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä –≤—Å—Ç—Ä–µ—Ç–∏—Ç 0 –ø—Ä–æ–±–µ–ª–æ–≤, —ç—Ç–æ –±—É–¥–µ—Ç –∑–Ω–∞—á–∏—Ç—å, —á—Ç–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ –∑–∞–∫–æ–Ω—á–µ–Ω–æ
    if a <= 0:
        # [8 –ø—Ä–æ–±–µ–ª–æ–≤] - –≤–Ω—É—Ç—Ä–∏ —Ñ—É–Ω–∫—Ü–∏–∏ —Ç–∞–∫ –∂–µ –º–æ–≥—É—Ç –±—ã—Ç—å –≤–ª–æ–∂–µ–Ω–Ω—ã–µ –±–ª–æ–∫–∏, –Ω–∞–ø—Ä–∏–º–µ—Ä —É—Å–ª–æ–≤–∏—è, —Ü–∏–∫–ª—ã –∏ –ø—Ä–æ—á–µ–µ
        return -a
    # [4 –ø—Ä–æ–±–µ–ª–∞] - –±–ª–æ–∫ if –∑–∞–∫–æ–Ω—á–∏–ª—Å—è, –º—ã –≤–µ—Ä–Ω—É–ª–∏—Å—å –Ω–∞ —É—Ä–æ–≤–µ–Ω—å –±–ª–æ–∫–∞ —Ñ—É–Ω–∫—Ü–∏–π
    else:
        # [8 –ø—Ä–æ–±–µ–ª–æ–≤] - –∫–æ–¥ –Ω–∏–∂–µ, –æ—Ç–Ω–æ—Å–∏—Ç—Å—è –∫ –±–ª–æ–∫—É else
        return a


# [0 –ø—Ä–æ–±–µ–ª–æ–≤] - –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ –∑–∞–∫–æ–Ω—á–µ–Ω–æ, –¥–∞–ª—å—à–µ –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏—è –Ω–∞ "–∫–æ—Ä–Ω–µ–≤–æ–º" —É—Ä–æ–≤–Ω–µ
print(get_abs(0))
```
